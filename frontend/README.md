# OffCampus Frontend

React + Vite frontend for the OffCampus Opportunity Portal.

## Features

- Login and signup flows
- Job matching dashboard with notifications toggle
- Resume upload and analysis UI

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

By default the frontend calls the backend at `http://localhost:5000`.

## Build

```bash
npm run build
```

## Notes

- Backend must be running for API calls to work.
- For environment variables, see [README.md](../README.md) in the project root.
